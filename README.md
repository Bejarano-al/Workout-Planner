# Workout-Planner
A Python code created to help with workout sessions that helps you create a personalized list of exercises based on what muscle group you want to train

# How It Works
-User selects a muscle group
-Chooses at least one main exercise
-Program auto-fills the rest using prioritized exercises
-Final workout is printed with details
